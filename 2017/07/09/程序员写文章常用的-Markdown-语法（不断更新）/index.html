程序员写文章常用的 Markdown 语法（不断更新）

最近将博客重新搭建了一下尊崇了简洁的原则，其中参考了 stormzhang 和 smartbeng 的博客，然后小小的调整了一下。

搭建完成后学习了一下 Markdown 的基本语法，顺便做个总结：

标题

标题文本前加 # ，加一个为一级标题，加两个为二级标题

eg：效果如下

## 二级标题

二级标题

列表

  • 无序列表

    只需要在文字前加上 - 或者 * 即可变为无序列表，文字和符号之间要加上一个空格。

    eg：效果如下

    * 无序列表1
    * 无序列表2

    • 无序列表1
    • 无序列表2

    ​

  • 有序列表

    有序列表与无序列表类似，知识每一行的前面换成了 1. 、2. …

    eg：效果如下

    1. 有序列表1

    2. 有序列表2

    1. 有序列表1
    2. 有序列表2

引用

如果文章中需要插入别处的文本，就要用到引用

>站在巨人的肩膀上才能看得更远

站在巨人的肩膀上才能看得更远

链接

有的地方需要插入链接，需要这样使用

[链接的文本](链接的地址)
